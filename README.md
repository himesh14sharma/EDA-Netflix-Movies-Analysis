# Netflix Movie Analysis

This project performs **exploratory data analysis (EDA)** on a Netflix movies dataset using **Python, Pandas, and NumPy**.  
The main goal is to extract insights and trends from Netflix's movie collection, such as genre distribution, release year trends, top-rated movies, and content availability across countries.

## Features / Highlights
- Data cleaning and preprocessing using **Pandas**  
- Statistical analysis with **NumPy**  
- Analysis of movie genres, ratings, and release years  
- Identification of patterns and trends in Netflix content  
- Visual insights (optional) using Pandas aggregation results  

## Dataset
- Dataset contains movie details such as title, genre, release year, rating, country, and duration  
- Publicly available or sample Netflix dataset  

## Tools / Libraries
- Python  
- Pandas  
- NumPy  

## Purpose
- Practice EDA using Pandas and NumPy  
- Understand Netflix content trends  
- Showcase data handling and analysis skills for GitHub portfolio
